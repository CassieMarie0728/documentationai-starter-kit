## OAuth Authentication Setup

Set up OAuth for supported services in minutes. MCP Hub handles token refresh and secure storage automatically.

<Steps>
  <Step title="Create OAuth App" icon="settings">
    In your service dashboard (e.g., GitHub Settings > Developer settings), create a new OAuth app. Set the callback URL to `https://api.mcphub.com/oauth/callback`.
  </Step>
  <Step title="Add Credentials" icon="key">
    Copy the Client ID and Client Secret to your MCP Hub workspace.
  </Step>
  <Step title="Authorize" icon="shield">
    Click "Connect" in MCP Hub. Complete the OAuth flow and grant permissions.
  </Step>
  <Step title="Test Connection" icon="check-circle">
    Run a test action from the integration dashboard to verify setup.
  </Step>
</Steps>

## Webhook Configuration

Configure outgoing webhooks to trigger MCP Hub workflows from external services. Incoming webhooks receive data securely.

<CodeGroup tabs="cURL,JavaScript,Python">
  ```bash
  curl -X POST https://api.mcphub.com/v1/webhooks/your-webhook-id \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "event": "user.signup",
      "data": {"user_id": 123, "email": "user@example.com"}
    }'
  ```
  ```javascript
  const response = await fetch('https://api.mcphub.com/v1/webhooks/your-webhook-id', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      event: 'user.signup',
      data: { user_id: 123, email: 'user@example.com' }
    })
  });
  ```
  ```python
  import requests

  response = requests.post(
      'https://api.mcphub.com/v1/webhooks/your-webhook-id',
      headers={
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
      },
      json={
          'event': 'user.signup',
          'data': {'user_id': 123, 'email': 'user@example.com'}
      }
  )
  ```
</CodeGroup>

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X POST https://api.mcphub.com/v1/webhooks/demo \
    -H "X-MCP-Signature: v1=abc123..." \
    -d '{"payload": "test"}'
  ```
  ```javascript
  // Verify webhook signature
  const crypto = require('crypto');
  const signature = 'sha256=' + crypto.createHmac('sha256', WEBHOOK_SECRET).update(payload).digest('hex');
  ```
</Request>

<Response tabs="200,400">
  ```json
  {
    "status": "success",
    "workflow_id": "wf_abc123",
    "execution_id": "exec_456def"
  }
  ```
  ```json
  {
    "error": "Invalid signature",
    "code": "WEBHOOK_SIGNATURE_MISMATCH"
  }
  ```
</Response>

<Callout kind="tip">
  Always verify webhook signatures using your secret to prevent unauthorized triggers. Find your webhook URL and secret in the MCP Hub dashboard.
</Callout>
