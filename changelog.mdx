## Recent Updates

<Callout kind="info">
  Stay current with the latest MCP Hub releases. Check this page regularly for new features, improvements, and fixes that enhance your automation workflows.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for Notion databases in workflows, enabling dynamic data pulls and updates.
- Introduced webhook triggers for instant workflow execution from external services.

## Improvements

- Enhanced dashboard performance with real-time metrics caching.
- Optimized token refresh logic for OAuth integrations.

## Bug Fixes

- Fixed intermittent failures in parallel execution branches.
- Resolved credential rotation issues during high-load scenarios.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Team workspaces now support shared macros with role-based permissions.
- Added audit logging for all workflow executions and credential access.

## Bug Fixes

- Corrected Slack notification delays in error handling.
- Patched GitHub API rate limit handling to prevent workflow stalls.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Updated workflow schema to version 2.0; migrate existing macros via the dashboard importer.

## New Features

- Core workflow engine with conditional logic and loops.
- Secure credential storage using AES-256 encryption.

## Initial Setup

- Production-ready dashboard at `https://dashboard.example.com`.
- API endpoints available at `https://api.example.com/v1/workflows`.

</Update>
